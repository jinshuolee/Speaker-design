# 使用類比電路製作攜帶式雙聲道喇叭

我們喇叭主要分成三個部分進行，分別為喇叭電路設計、單體製作和外盒設計。
1.單體製作:
單體原理為當線圈通過電流時會產生磁場，產生的磁場和磁鐵作用產生振動，市面上單體主要材料為懸邊、前導磁版、盆架、彈波、錐盆、音圈和中心磁極，而我們製作時使用較簡易的材料代替，使用磁鐵和線圈，並用海綿當成簡易的阻尼，大大降低了成本，纏繞兩組4歐姆阻抗的線圈，作為雙聲道的單體，最終可成功播放出音樂。
2.喇叭電路設計:
電路設計主要使用PAM8610雙聲道功率放大板，這個功放板在輸出給4歐姆阻抗的喇叭時，最高輸出功率單聲道可達到15W。旋鈕連接一個可變電阻，可以調整整體音量的大小，開關按鈕和LEC燈串連電源，當按下開關時LED燈會亮起。電源提供兩種輸入方式，可以使用12V、2A的變壓器，或是3顆3.7V的電池供電，添加一個鋰電池保護板來防止電池過放電和短路保護。
3.外盒設計:
使用Solidworks軟體繪製外盒圖，用雷射切割密集板來加工，設計一個美觀的外盒。喇叭前方為單體和網罩，上方為電源開關和音量旋鈕，後方為電源和音源線輸入，內部另外添加吸音棉來吸音降躁。
